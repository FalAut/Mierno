# 如何启动/运行服务器

**当然你可以使用自己的方式来启动/运行服务器**

## 准备

在 variables.txt 中找到 JAVA="path/to/java"
将 "path/to/java" 改为你的 Java 路径，比如 "C:/Program Files/Java/jdk-17/bin/java.exe"
注意路径分割号需要是 / 而不是 \
如果没有设置指定的路径，可能会尝试使用系统默认的 Java（如果存在的话）
完成后根据你的系统类型阅读下方的说明

## Windows

运行 `start.bat` 启动服务器。
**不要删除 PowerShell (ps1) 文件！**

您可以从控制台窗口手动运行 `start.ps1`，但建议使用批处理脚本。
运行 PowerShell 脚本需要更改 Windows 系统的执行策略。批处理脚本
可以绕过启动脚本。

提示：start.bat 比 start.ps1 更好

## Linux

运行 `.\start.sh` 或 `bash start.sh` 启动服务器。

## MacOS

运行 `.\start.sh` 或 `bash start.sh` 启动服务器。

## 注意

**当你在控制台中看到这段信息**

_Mojang's EULA has not yet been accepted. In order to run a Minecraft server, you must accept Mojang's EULA._
_Mojang's EULA is available to read at https://aka.ms/MinecraftEULA_
_If you agree to Mojang's EULA then type 'I agree'_
_Answer:_

**此时请在控制台输入 I agree 来同意 Mojang 的 EULA，然后才能继续启动服务器**
**在这之后会开始下载服务端所需的库，请耐心等待并保持网络通畅**
**遇到问题可前往社群获取帮助**

Forge 和 NeoForge 1.17 及以上版本会创建 run.xx 脚本，这是因为 ServerStarterJar 用于安装和运行服务器。
忽略这些脚本并继续使用 start.xx 脚本是安全的。
删除 run.xx 脚本将导致服务器再次由 ServerStarterJar 安装。有关
ServerStarterJar 的更多信息，请访问 https://github.com/neoforged/ServerStarterJar。
